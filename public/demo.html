Demo Trang Web với Chatbot Nhúng

Đây là một trang web demo để test chatbot nhúng.

Chatbot sẽ xuất hiện ở góc dưới phải của trang.

Click vào icon 💬 để mở chat.

Cách nhúng chatbot vào trang web của bạn:

  1. Upload file chatbot.js từ thư mục dist/ lên host của bạn
  2. Thêm dòng sau vào cuối thẻ <body> của trang HTML: 
<!-- Load Chatbot Script -->
<script src="path/to/chatbot.js"></script>

Sau khi thêm, chatbot sẽ tự động xuất hiện và hoạt động! CSS đã được tích hợp trong script.
